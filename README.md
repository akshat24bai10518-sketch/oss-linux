## Project Description
This project is a structured audit of the VLC Media Player open-source project. [cite_start]It includes a detailed report on its origins at École Centrale Paris [cite: 40][cite_start], its GPL licensing [cite: 24][cite_start], and five shell scripts that demonstrate Linux system administration and automation skills[cite: 89, 90].

## Shell Scripts Included
1. [cite_start]**script1_identity.sh**: Generates a system identity report including kernel and distro info[cite: 93].
2. [cite_start]**script2_inspector.sh**: Checks for VLC installation and its open-source philosophy[cite: 125].
3. [cite_start]**script3_auditor.sh**: Audits system directory permissions and disk usage[cite: 145].
4. [cite_start]**script4_logs.sh**: Analyzes system logs for specific error keywords[cite: 163].
5. [cite_start]**script5_manifesto.sh**: An interactive script to generate a personalized OSS manifesto[cite: 185].

## How to Run

2. Change directory: `cd java-expense-tracker`
3. Make scripts executable: `chmod +x *.sh`
